# clean-architecture-practice

クリーンアーキテクチャーを体験するためのリポジトリです。

## 始め方
### 1.リポジトリをforkする  
▼forkの仕方  
https://docs.github.com/ja/get-started/quickstart/fork-a-repo

### 2.モジュールのダウンロード  
```
go mod download
```
で関連モジュールのダウンロードをする

### 3.サーバーを立ち上げる
```
docker-compose up -d
```
でサーバーを立ち上げる  
`localhost:8080`でアクセスできます

### 4.練習問題を解く  
`practice` ディレクトリに問題が入っているのでそれを解いていってください
